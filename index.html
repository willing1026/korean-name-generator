About This Korean Name Generator

This free Korean name generator converts any English name into realistic Korean names that are actually used in Korea. Unlike simple transliteration tools, our generator suggests names based on the top 300 most popular Korean baby names from the last 3 years, giving more weight to recent trends.

Why Use a Korean Name?

Many foreigners choose a Korean name when living in Korea, studying the language, or making Korean friends online. A proper Korean name helps with pronunciation, cultural integration, and social connection. Our tool provides both family name + given name options, as well as kept-form versions (e.g. "Liam → 리암").

How It Works

  • Type an English name
  • Select gender (male, female, unisex, or auto-detect)
  • Get realistic Korean names with pronunciation (romanization)
